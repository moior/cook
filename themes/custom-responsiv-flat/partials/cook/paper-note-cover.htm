표지 노트의 표지

표지 스타일 가장 많이 쓰는 2가지 방식 중 골라주세요

일반 인쇄

스노우/아트지에 전면 인쇄합니다.
예제 {% for i in 2..6 %} {% if 1 %} {% endif %} {% endfor %}
표지의 안쪽까지 인쇄하려면 앞뒤 인쇄를 선택해주세요.

금은박 디자인

깔끔한 바탕에 금은박으로 디자인
예제 {% for i in 1..5 %} {% if 1 %} {% endif %} {% endfor %}
표지의 두께 표지 제작방식. 단단함에 차이가 있음
제본방식에 따라 선택할 수 있는 표지 방식이 제한됩니다. (예, 하드커버는 스프링제본에서만 가능)

금은먹박 회사 로고 등
